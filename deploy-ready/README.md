## Deployment

This website is optimized for Netlify deployment.

### Files Structure
```
deploy-ready/
- index.html          # Main HTML file
- styles.css          # Complete styling
- script.js           # Interactive functionality
- logo.png            # Company logo
- netlify.toml        # Netlify configuration
- _redirects          # URL routing rules
- README.md           # This file
```

### Deployment Instructions

1. **Delete existing Netlify site** (if any)
2. **Add new site** on Netlify dashboard
3. **Deploy manually** by dragging all files from `deploy-ready/` folder
4. **Or use GitHub import** for automatic deployment

### Netlify Configuration

- **Build Command**: Not required (static site)
- **Publish Directory**: Root directory
- **Node Version**: 18 (specified in netlify.toml)
